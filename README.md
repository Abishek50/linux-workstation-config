## Customization Highlights

### Shell and System Identity
- Customized neofetch ASCII logo
- Enhanced shell environment using zsh
- ASCII banner tooling via figlet and toilet

### Dynamic Theming
- Integrated pywal with ImageMagick
- Terminal color scheme updates automatically based on wallpaper
- Alias-based workflow for manual refresh

### File Manager Automation
Implemented a custom script to copy absolute paths of one or more files
selected in the Nemo file manager.

Features:
- Handles multiple file selections
- Strips file:// prefixes
- Copies results directly to clipboard using xclip
- Sends desktop notifications on completion

The script is installed in ~/.local/bin to ensure availability in user PATH.
